Dashboard Pulse

1. De la Investigación a la Operación

El descubrimiento de clústeres y reglas de asociación de alto valor en entornos notebooks es esencial, pero carece de utilidad operativa si los equipos comerciales no pueden interactuar con los resultados. Para cerrar este puente, desarrollamos la plataforma Pulse Dashboard.

2. Arquitectura de Pulse

Pulse es una aplicación analítica construida con una arquitectura moderna de microservicios:

  • Backend / API: Desarrollado utilizando un stack ágil (como FastAPI o Flask) expuesto a través de enrutadores dedicados (routers/).
  • Capa de Datos Analítica (db.py & queries.py): Interfaz optimizada que alimenta el dashboard con los datos procesados durante el modelado (archivos .parquet de alta eficiencia y/o un almacén SQL).
  • Frontend Interactivo (templates/ y static/): Sistema de vistas que presenta a Marketing y Ventas la información procesada.

3. Funcionalidades Principales de la Interfaz

El dashboard no requiere que el usuario entienda matemáticas complejas o Machine Learning. Se enfoca en la accionabilidad directa:

  1. Visor de Salud del Negocio: Una pantalla inicial que muestra la distribución actual de los segmentos. ¿Qué porcentaje de nuestros clientes ha caído a estado Hibernando este mes?
  2. Generador de Bundles (MBA): Seleccionando un segmento específico (ej. MVPs), el dashboard consulta las reglas exclusivas extraídas y expone las mejores combinaciones de productos para lanzar una campaña.
  3. Mapas de Calor Activos: Gráficas basadas en temporalidad_segmento_hora_dia.parquet que indican en verde fuerte las horas exactas en las que se debe lanzar una campaña de email para un segmento en particular.

4. Impacto

La integración del pipeline automatizado y la interfaz Pulse cambia por completo la estrategia de CT Internacional. Ahora, en lugar de solicitar un presupuesto de marca sin respaldo, el equipo puede argumentar: “Para este producto, necesitamos X presupuesto y atacaremos al segmento Alto Valor a las 11:00 AM del martes con este bundle específico, el cual sabemos que tiene un Lift de 40x”.
